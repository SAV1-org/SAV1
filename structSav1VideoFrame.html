Detailed Description

Struct to represent one decoded video frame.

Contains packed pixel data in the format of the user's choice, as well as metadata to aid in its usage. Ownership of Sav1VideoFrames belongs to SAV1, unless the user calls the sav1_video_frame_take_ownership function, in which case they are responsible for calling the sav1_video_frame_destroy function to destroy it.
