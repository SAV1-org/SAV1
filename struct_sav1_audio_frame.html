Detailed Description

Struct to represent one decoded audio frame.

Contains raw audio data, as well as metadata to aid in its usage. Ownership of Sav1AudioFrames belongs to SAV1, unless the user calls the sav1_audio_frame_take_ownership function, in which case they are responsible for calling the sav1_audio_frame_destroy function to destroy it.
