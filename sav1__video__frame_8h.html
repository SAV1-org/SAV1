typedef struct Sav1VideoFrame Sav1VideoFrame

Struct to represent one decoded video frame.

Contains packed pixel data in the format of the user's choice, as well as metadata to aid in its usage. Ownership of Sav1VideoFrames belongs to SAV1, unless the user calls the sav1_video_frame_take_ownership function, in which case they are responsible for calling the sav1_video_frame_destroy function to destroy it.

SAV1_API int sav1_video_frame_clone ( Sav1Context *  context,
Sav1VideoFrame *  src_frame,
Sav1VideoFrame **  dst_frame 
)

Deep clone an existing Sav1VideoFrame.

SAV1 will allocate a new video frame and new data within to copy the provided frame. SAV1 will not hold ownership of the newly created frame, so the user is responsible for calling the sav1_video_frame_destroy function when they are done with it.

If the user is using Sav1Settings::custom_video_frame_processing, the Sav1VideoFrame::custom_data member will only be shallow-cloned.

Parameters
[in]contextpointer to a created SAV1 context
[in]src_framepointer to a Sav1VideoFrame to be cloned
[out]dst_framepointer to the new Sav1VideoFrame
Returns
0 on success, or < 0 on error

Free memory used by a Sav1VideoFrame.

Normally, SAV1 maintains ownership over Sav1VideoFrame`s and is responsible for destroying them. If the user has ownership over a frame, either by calling the sav1_video_frame_take_ownership function to claim ownership of an existing frame, or the sav1_video_frame_clone function to create a new clone of an existing frame, then they must call this function to free its memory after the frame is no longer needed.

Parameters
[in]contextpointer to a created SAV1 context
[in]framepointer to a Sav1VideoFrame that the user has ownership of

0 on success, or < 0 on error
